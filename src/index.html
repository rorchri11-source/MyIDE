MyIDE — Guida Rapida

Primi Passi

  1. Clicca Impostazioni (⚙️) in cima alla pagina
  2. Vai su Providers → + Add Provider
  3. Scegli un template (OpenRouter o Google AI Studio)
  4. Inserisci la tua API Key
  5. Salva e seleziona il provider dalla barra in alto

Modalita

  • Chat — conversazione libera
  • Edit — l'AI legge e modifica il file aperto
  • Agent — l'AI lavora su tutto il progetto

Scorciatoie

  • Ctrl+Enter Invia · Ctrl+S Salva · Ctrl+Shift+N Nuova Chat
  • Ctrl+Shift+T Terminale · Ctrl+Shift+E Sidebar
